# JavaScript API Web App

A web application built with **HTML, CSS and JavaScript** that fetches data from a **public API** and displays it dynamically on the page.

The project demonstrates how to work with **external APIs, asynchronous JavaScript, and DOM manipulation** to create an interactive frontend application.

---

# 🚀 Features

- Fetch data from a public API
- Dynamic content rendering
- Asynchronous JavaScript using `fetch`
- Interactive user interface
- Error handling for API requests

---

# 🧱 Technologies

- HTML
- CSS
- JavaScript
- Public API

---

# 🎯 Learning Goals

This project focuses on learning and practicing:

- Working with APIs
- Fetching external data
- Asynchronous JavaScript (`fetch`, `async/await`)
- DOM manipulation
- Handling API responses

---

# ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Flizzyar/javascript-api-app.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

No installation or build step is required.

---

# 📡 API Usage

The application fetches data from a public API and displays it on the page.

Example workflow:

1. A request is sent using `fetch()`.
2. The API returns JSON data.
3. The data is parsed and displayed dynamically on the page.
